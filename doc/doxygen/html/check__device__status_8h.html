◆ check_wifi_state()

int check_wifi_state ( char * interface)

检查WIFI的工作状态

该函数通过读取系统文件来检查指定WIFI接口的工作状态。 如果接口状态为"up"，则表示WIFI已连接；否则表示WIFI未连接。

参数
{char*} interface WIFI接口名称
返回
0 表示WIFI已连接，1 表示WIFI未连接或读取失败
