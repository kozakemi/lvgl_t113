\begin{DoxyParagraph}{功能描述}

\begin{DoxyEnumerate}
\item 初始化\+LVGL库
\item 配置显示驱动（sunxifb）
\item 配置输入设备驱动（evdev）
\item 提供命令行参数选择不同的演示模式或启动自定义页面（主页、日历页、消息页）
\item 启动\+MQTT客户端用于网络通信
\end{DoxyEnumerate}
\end{DoxyParagraph}
